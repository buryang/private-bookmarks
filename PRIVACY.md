# Privacy Policy for Private Bookmarks Chrome Extension

**Last updated: March 11, 2026**

Private Bookmarks is a fully open-source Chrome extension designed for local, private bookmark management.

**We do NOT collect, transmit, use, share, or sell any user data.**

- All data (including hidden bookmarks, password hash, AI-generated tags, and user settings) is stored **locally on your device** using Chrome's storage API.
- No information is sent to any server owned by us or third parties.
- If you enable **configurable cloud sync**, it uses your chosen end-to-end encrypted service (e.g., WebDAV, Supabase, self-hosted). We do not control, access, or see your sync data — it's fully user-managed and encrypted.
- No telemetry, analytics, cookies, tracking, or logging of any kind.
- The extension complies with Chrome Manifest V3 and does not request unnecessary permissions.

For source code and transparency: https://github.com/[你的用户名]/private-bookmarks

If you have questions, open an issue on GitHub.

This policy may be updated; changes will be posted here.
